1. ### 编辑推送/催办事项

1. **URL**: `/{strategyType}/communicate/event/edit`
2. **方法**: `POST`
3. **权限**: `communicateEventEdit`

请求参数:

```JSON
{
  "eventCode": "EVT_20250918_0001",
  "taskCode": "TASK_001",
  "pushObject": [],
  "param": {
      "filter": [
          {
              "title": "string", // 筛选标题
              "value": "string", // 筛选值
              "operator": "string", // 筛选操作符
              "type": "string" // 筛选组件类型
          },
      ]
      "isAnd": "and" // true：且，false：或
  },
  "pushTime": [],
  "templateContent": "推送正文"
}
```

响应信息:

```JSON
{
  "code": 500,
  "msg": "失败：具体错误信息",
  "data": null
}
```
