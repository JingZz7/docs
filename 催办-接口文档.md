1. ### 编辑推送/催办事项

- **URL**: `/{strategyType}/communicate/event/edit`
- **方法**: `POST`
- **权限**: `communicateEventEdit`

请求参数:

```JSON
{
  "eventName": "事项A",
  "eventType": "PUSH/URGE",
  "taskCode": "TASK_001",
  "pushObject": ["OTDTeam","BPList","UnFinishedTaskManagers"],
  "param": {
      "filter": [
          {
              "code": "",
              "key": "",
              "value": "111"
          },
          {
              "code": "",
              "key": "",
              "value": "222"
          }
      ]
      "logicRelation": "and"
  },
  "pushTime": {
      "timeGranularity": "day", // 每天-day，每周-week，每月-month，bp消息推送后-bpPushed，管理者消息推送后-managePushed，任务截止时间前-beforeTask Deadline，具体时间-Data
      "specificGranularity": "",
      "timePointGranularity": ""
  },
  "content": "推送正文"
}
```

响应信息:

```JSON
{
  "code": 500,
  "msg": "失败：具体错误信息",
  "data": null
}
```
